This repository contains reproducibility materials for the manuscript ""Facial appearance and institutional cues in political trust evaluations".

The uploaded files include:

  • study1.csv Data for study / experiment 1
  • study2.csv Data for study / experiment 2
  • study3.csv Data for study / experiment 3
  • study4.csv Data for study / experiment 4
  • facexinst_bora.R R code with all the analysis.
  • summarySE.R A custom R function referenced in the code.

You need to download and extract the files, and specify the setwd() funtion in the R script in order to reproduce my findings.
